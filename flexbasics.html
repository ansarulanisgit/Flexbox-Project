CSS Flexbox -- is CSS3 Web Layout Module --Parent Element need to "display: flex" Flex Parent Container Properties: -- flex-direction: row, row-reverse, column, column-reverse; -- flex-wrap: nowrap, wrap, wrap-reverse; -- flex-flow: row wrap, column wrap-reverse; (addition of flex-direction & flex-flow) -- justify-cotent: flex-start, center, flex-end, space-between, space-around; (align childs horizontaly) -- align-items: flex-start, center, flex-end, stretch, baseline; (align childs vertically) Note: justify-content & align-items both are use to centralize any element in a parent element -- align-content: flex-start, center, flex-end, space-between, space-around, stretch; Flexbox Child Elements Properties: -- flex: number; -- flex-basis: number/percentage; -- flex: number number/percentage; (Addition of flex & flex-basis) -- align-self: flex-start, center, flex-end; -- order: number; -- flex-grow: number; -- flex-shrink: number;
